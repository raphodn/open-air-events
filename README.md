# CinéPleinAir

Un petit projet pour connaître les projections de films en plein air autour de chez soi. Collaboratif. Basé sur [OpenStreetMap](https://www.openstreetmap.org) & [OpenEventDatabase](https://wiki.openstreetmap.org/wiki/OpenEventDatabase).

à compléter

## Stack technique

* Vue.js 3 + Vuetify 3
* OpenEventDatabase pour stocker et synchroniser les événements
